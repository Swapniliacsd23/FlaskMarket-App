# FlaskMarket

A gadget market web application built with Flask.

## Features

- User authentication
- Buy & sell items
- Budget tracking
- Admin tools

## Tech Stack

- Python + Flask
- SQLAlchemy
- Flask-WTF
- Flask-Bcrypt
- Flask-Migrate
- Bootstrap 5

## How to Run

1. Clone the repo  
2. Create a virtual environment  
3. Install requirements: `pip install -r requirements.txt`  
4. Run with: `python run.py`
